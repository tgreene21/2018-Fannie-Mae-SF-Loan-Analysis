# 2018 Fannie Mae SF Loan Analysis

This Jupyter Notebook analyzes SF loans dataset in order to illustrate working with Python Data Analysis Tools. I will be examining the Purchase Price of the loan and how it relates to a few different variables.

</br>This project utilizes Pandas/Numpy for data cleaning, Seaborn for visualizations, and Sci-kit learn for ML

</br> Click the <a href = "https://github.com/tgreene21/2018-Fannie-Mae-SF-Loan-Analysis/blob/main/main.ipynb"> main.ipynb </a> to get started
</br>For the dataset please see: fhlmc_sf2018d_loans.txt
</br>For explanations on the Dataset, please see: 2018_Single_Family_National_File_C.pdf
</br>Source: https://www.kaggle.com/datasets/jeromeblanchet/fannie-mae-freddie-mac-public-use-database
